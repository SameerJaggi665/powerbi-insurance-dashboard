# 🛡️ Insurance Claims & Premium Analysis Dashboard (Power BI)

## 📊 Project Overview

This project presents an interactive **Insurance Analytics Dashboard** built using **Power BI** to analyze policy performance, claims distribution, and customer segmentation.

The dashboard provides insights into:

* Total **Premium Amount**
* Total **Coverage Amount**
* Total **Claim Amount**
* Claim status trends (Rejected, Settled, Pending)
* Policy performance by type
* Customer demographics (gender & age group)
* Active vs Inactive policies

## 🔑 Key Insights

### 💰 Financial Overview

* **Total Premium Amount:** 5.98M
* **Total Coverage Amount:** 600.55M
* **Total Claim Amount:** 16.91M

👉 Coverage significantly exceeds claims, indicating strong underwriting margins.

---

### 📌 Claims Analysis

* **Rejected Claims:** ~4.4K
* **Settled Claims:** ~3.4K
* **Pending Claims:** ~2.3K

👉 A high number of rejected claims may indicate:

* Strict validation policies
* Potential customer dissatisfaction risks

### 🧾 Policy Type Performance

* **Top Performing Policy:** Travel (~2.5M premium)
* Followed by:

👉 Travel insurance generates the highest premium revenue.
